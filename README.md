# 通过 php-webdriver (Selenium) 进行视频文件嗅探的例子

## 安装

1. PHP-CLI
2. Chrome
3. [Chrome Driver](https://sites.google.com/a/chromium.org/chromedriver/downloads)

## 配置

打开`run.php`，修改`Chrome driver`的路径。

## 运行

```
php run.php 2>&1 | tee run.log
```

## 结果

查看 [run.log](run.log)。
